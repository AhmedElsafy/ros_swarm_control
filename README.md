# ros_swarm_control
The ROS package to control the swarm of robots on the base px4 with data transmission through [Rosbridge](http://wiki.ros.org/rosbridge_suite).

# Branch:
* rc_car (for [rc_car_ros](https://github.com/GigaFlopsis/rc_car_ros))
